Selectores descendientes

LOS SELECTORES DESCENDIENTES INDICA CUANDO UN ELEMENTO ES HIJO, NIETO, BISNIETO, ETC. DE OTRO, EN RESUMEN ES UN ELEMENTO QUE ESTÁ DENTRO DE OTRO NO IMPORTA EL NIVEL.

DEL SIGUIENTE CÓDIGO TODOS LOS ELEMENTOS li SON DESCENDIENTES (HIJOS) DEL ul

            <ul class="list">
              <li class="list-item">Item 1</li>
              <li class="list-item">Item 2</li>
              <li class="list-item">Item 3</li>
              <li class="list-item">Item 4</li>
            </ul>

SI QUEREMOS QUE TODOS LOS ITEMS (QUE SON HIJOS DE ul) TENGAN EL COLOR ROJO DEBEMOS USAR UN SELECTOR DESCENDIENTE EN CSS, EL SIGUIENTE CÓDIGO CSS NOS SIRVE PARA ESE FIN: 

                .list .list-item { /* CON ESTE CÓDIGO LE ESTAMOS DICIENDO QUE A TODOS LOS .list-item QUE SEAN HIJOS DE .list les APLIQUE EL COLOR ROJO.
                    color: red;    LOS ELEMENTOS PADRE VAN A LA IZQUIERDA SEPARADOS POR UN ESPACIO DE LOS HIJOS QUE VAN A LA DERECHA, SI LE QUITAMOS EL ESPACIO.
                }                  NO FUNCIONARÁ COMO QUEREMOS. */
                                   /*NOTA: No se recomienda usar mas de 3 niveles en el uso de selectores descendientes */

NOTA: LAS BUENAS PRACTICAS EN CSS RECOMIENDAN QUE NO SE USEN SELECTORES DESCENDIENTES SALVO QUE DE VERDAD NO TENGAMOS OTRA OPCIÓN.

ESTE ES EL CÓDIGO CSS QUE TENEMOS QUE USAR PARA PONER EL COLOR AZUL A LOS HIJOS DIRECTOS DEL ul ANIDADO

Selector hermano siguiente

UN HERMANO SIGUIENTE ES EL ELEMENTO QUE ESTÁ JUSTO DESPUÉS DE OTRO.

EL SIGUIENTE CÓDIGO SE USARÁ PARA EJEMPLIFICAR EL USO DE LOS SELECTORES HERMANO SIGUIENTE. 

                <h1>Título</h1>
                <p class="subtitle">Contenido de h1</p>
                <h2>Subtítulo</h2>
                <p class="subtitle">Contenido de h2</p>

TENEMOS UN h1, UN p, UN h2 Y OTRO p COMO PODEMOS VER TODOS ESTOS ELEMENTOS SON HERMANOS, ENTONCES QUEREMOS QUE TANTO EL CONTENIDO DE h1 Y h2 TENGAN DIFERENTE COLOR COMO VEMOS ESTOS SON ELEMENTOS p Y AMBOS TIENEN LA CLASE contenido, ENTONCES SI QUEREMOS LOGRAR ESTO TENDRÍAMOS QUE AGREGAR UNA CLASE A CADA UNO Y CAMBIAR EL COLOR USANDO ESA CLASE, ESTO PODRÍA SER UNA POSIBLE SOLUCIÓN, PERO UNA MEJOR SERÍA USAR LOS SELECTORES HERMANOS, ¿PERO COMO SE USAN ESTOS EN CSS?

ESTE ES EL CÓDIGO CSS QUE TENEMOS QUE USAR PARA PONER UN COLOR DIOFERENTE A CADA ELEMENTO p DEL CONTENIDO. 

                .title + .contenido { /* PARA ESPECIFICAR QUE QUEREMOS PONERLE ESTILO A UN ELEMENTO HERMANO SIGUIENTE SE USA EL + */
                    color: aqua;
                }
                    
                .subtitle + .contenido {
                    color: green;
                }

Selector de atributo

UN SELECTOR DE ATRIBUTO SELECCIONA UN ELEMENTO POR CUALQUIER ATRIBUTO QUE NO SEA NECESARIAMENTE UN CLASE O UN ID, TODOS LOS ATRIBUTOS SE ENCIERRAN ENTRE [], SU SINTAXIS ES [attr="value"] PARA ATRIBUTO Y VALOR, [attr]. PARA SOLO ATRIBUTO, TAMBIEN SE PUEDEN USAR LOS COMODINES:
^: EMPIEZA CON [attr^="value"]
$: TERMINA CON [attr$="value"]
*: CONTIENE [attr*="value"]
~: CONTIENE (SEPARADOS POR ESPACIOS) [attr~="value"]
|: COMIENZA CON (MAS GUIÓN) [attr|="value"]
ESTOS SELECTORES SE USAN MÁS EN FORMULARIOS CON LOS ELEMENTOS input

ESTE ES EL CÓDIGO HTML QUE TENEMOS QUE USAR PARA ESTE EJEMPLO DONDE QUEREMOS ESTABLECER ESILOS DIFERENTES PARA LOS INPUT, PERO PARA LOGRARLO LO MÁS FACIL TAL VEZ SERÍA PONERLE UNA CLASE A CADA ELEMENTO Y EN BASE A ELLA PONERLE UN ESTILO, PERO PARA EVITARNOS ESO ESTAN LOS SELECTORES DE ATRIBUTO. 

                <form>
                  <input type="mail">
                  <input type="submit" value="Enviar">
                </form>

ESTE ES EL CÓDIGO CSS QUE USAREMOS PARA REALIZAR EL EJERCICIO

                    [type="mail"] {
                        background-color: #fff;
                    }
                    
                    [type='submit'] {
                        background-color: blue;
                        color: #fff;
                    }
                    
                    [required] {
                        border: 2px solid red;
                        margin: .5em;
                    }

Selector de atributo comienza con

ESTE SELECTOR DETECTA CUANDO EL ATRIBUTO COMIENZA CON ALGO, EJEMPLO TENEMOS 3 ENLACES, DE LOS CUALES DOS APUNTAN A RECURSOS LOCALES Y UNO A UN EXTERNO, ENTONCES NECESITAMOS QUE LOS QUE APUNTEN A LOCAL TENGAN UN COLOR ROJO Y EL EXTERNO TENGA UN ICONO.

ESTE ES EL CÓDIGO HTML QUE USAREMOS PARA EL EJEMPLO

                <a href="/contacto.html">Contacto</a>
                <a href="https://dominio.com">Nosotros</a>
                <a href="https://youtube.com">Canal</a>

ESTE ES EL CODIGO CSS QUE USAREMOS PARA EL EJEMPLO

                    a {
                        background: url(../img/external_link.svg) no-repeat center right / 1rem;/* CON ESTO PONEMOS LA IMAGEN QUE ESTA AL LADO DEL ENLACE*/
                        padding-right: 1.5rem;    
                    }
                    
                    [href^="/"], [href^="https://dominio.com"] {/* PARA ESTABLECER EL COLOR Y EL BACKGROUND DE LOS ENLACES LOCALES USANDO EL SELECTOR TERMINA CON SE USA EL ^ 
                        color: red;                             TAMBIEN PODEMOS USAR UN DOMINIO COMPLETO PARA LA OPCIÓN DE COMIENZA CON.
                        background: none;                       EN ESTE EJEMPLO ADEMÁS DE USAR SELECTORES DE ATRIBUTO COMIENZA CON, ESTAMOS USANDO SELECTORES AGRUPADOS (SON
                    }                                           LOS QUE SE SEPARAN POR UNA COMA (,)). */
Contacto Nosotros Canal

Selectores de atributo termina con

ESTE SELECTOR DETECTA CUANDO EL ATRIBUTO TERMINA CON ALGO, EJEMPLO TENEMOS 2 ENLACES, DE LOS CUALES UNO TERNIMA CON LA EXTENSIÓN .pdf, ENTONCES QUEREMOS QUE ESTE TENGA UN ICONO DE pdf

ESTE ES EL CÓDIGO HTML QUE USAREMOS EN EL EJMPLO

                <a href="/clase1.html">Clase 1</a>
                <a href="/recursos/clase1.pdf">Notas</a>

                    [href$="pdf"] { /* PARA ESTEBLECER ESTILOS USANDO EL SELECTOR TERMINA CON SE USA EL $, EN ESTE CASO ESTEMOS ESPECIFICANDO QUE LOS ATRIBUTOS QUE 
                                    TERMINEN CON .pdf TENDRÁN UN ICONO. */
                        background: url(../img/pdf-icon.svg) no-repeat right / 1.3rem;    
                    }

Selector de atributo contiene

EL SELECTOR CONTIENE, HACE REFERENCIA A QUE EL PATRÓN QUE BUSCAMOS PODRÍA ESTAR EN CAULQUIER PARTE DEL ATRIBUTO. EJEMPLO TENENOS DOS ENLACES Y QUEREMOS DARLE CIERTOS ESTILOS IGUALES, ES CIERTO QUE PODRÍAMOS USAR CLASES IGUALES PARA LOS ELEMENTOS Y EN BASE A ESO ESTABLECER LOS ESTILOS, PERO COMO SOLO SERÁN ALGUNOS Y ADEMÁS DE ESOS TENDRÁN OTROS, SERÍA TEDIOSO PONERLE MUCHAS CLASES, ES EN SITUACIONES COMO ESTA QUE PODEMOS USAR ESTE SELECTOR.

ESTE ES EL CÓDIGO HTML QUE USAREMOS PARA EL EJEMPLO

                <a href="#" class="btn-sign-in">Sign In</a>
                <a href="#" class="btn-sign-up">Sign up</a>

ESTE ES EL CÓDIGO CSS QUE USAREMOS PARA EL EJEMPLO

                    [class*="btn-"] { /* PARA ESTABLECER ESTILOS CON EL SELECTOR CONTIENE SE USA EL *, EN ESTE CASO ESTAMOS ESPECIFICANDO QUE A LOS ELEMENTOS QUE TENGAN EN EL
                                         ATRIBUTO CLASS EL PATRÓN btn- SE LES ESTABLEZCAN LAS PROPIEDADES ABAJO ANOTADAS, COMO VEMOS ESTOS ESTILOS SON COMPARTIDOS POR LOS 
                                         ENLACES SIN NECESIDAD DE PONERLE MUCHAS CLASES, SI NO SOLO USANDO UNA*/
                        display: inline-block;
                        text-decoration: none;
                        padding: 1rem 2rem;
                        border: 1px solid grey;
                        widows: 2em;
                        border-radius: .3em;
                    }
                    
                    /* DESPUÉS ESTABLEZEMOS ESTILOS INDIVIDUALES EN BASE A LOS NOMBRE DE LA CLASE. */
                    .btn-sign-in {
                        background-color: cornflowerblue;
                    }
                        
                    .btn-sign-up {
                        background-color: green;
                    }

Selector de atributo contine (seperado por espacios)

EL SELECTOR CONTIENE (SEPARADO POR ESPACIOS) HACE REFERENCIA A QUE EL ATRIBUTO TENDRÁ UN VALOR SEPARADO DE OTRO POR UN ESPACIO. EJEMPLO TENEMOS DOS BOTONES CON UNA CLASE btn-link Y QUEREMOS QUE COMPARTAN ALGUNOS ESTILOS, PERO NO TODOS, ENTONCES PODRÍAMOS PONER UNA CLASE EXTRA A UNO DE ELLOS Y USAR ESA, YA SEA DE MANERA INDEPENDIENTE (USANDO UN SELECTOR DE CLASE) O PODRÍAMOS USAR UN SELECTOR DE ATRIBUTO CONTIENE (SEPARADO POR ESPACIOS).

ESTE ES EL CÓDIGO HTML USADO PARA EL EJEMPLO

                <a href="#" class="btn-link">Ir..</a>
                <a href="#" class="btn btn-link">Regresar</a>

ESTE ES EL CÓDIGO CSS USADO PARA EL EJEMPLO

                    [class~="btn"] { /* PARA ESTABLECER ESTILOS USANDO EL SELECTOR CONTIENE (SEPARADO POR ESPACIOS) SE USA LA ~, Y PARA QUE TOME LOS ESTILOS BASTA CON SOLO 
                                        ESPECIFICAR UNA DE LAS CLASES, AQUÍ ESTAMOS USANDO LA PRIMERA (btn), PERO TAMBIEN PODRÍAMOS USAR LA SEGUNDA (btn-link). 
                        background-color: red;
                        color: #fff;
                    }

ESTE ES EL RESULTADO DEL EJEMPLO

Ir.. Regresar
NOTA: ESTE ELECTOR DE ATRIBUTO ES MUY POCO USADO

Selector de atributo comienza con (más guion)

EL SELECTOR COMIENZA CON (MAS GUIÓN) HACE REFERENCIA A QUE EL ATRIBUTO TENDRA UN PATRON AL INICIO, PERO LUEGO DE ESE PATRÓN VIENE UN GUIÓN. EJEMPLO TENEMOS UN PARRAFO EL CUAL TIENE LA CLASE texto-grande AL CUAL QUEREMOS APLICARLE UN TAMAÑO DE FUENTE MAS GRANDE.

ESTE ES ELCODIGO QUE USAREMOS PARA EL EJEMPLO

                <p class="texto-grande">Este texto tendrá un font size más grande que el resto.</p>

ESTE ES EL CODIGO CSS USADO PARA EL EJEMPLO

                    [class|="texto"] { /* PARA ESTABLECER ESTILOS USANDO EL SELECTOR COMIENZA CON (MÁS GUION) SE USA EL |, Y PARA QUE TOME LOS ESTILOS BASTA CON SOLO PONER
                                          EL INICIO EN ESTE CASO DE LA CLASE, NO ES NECESARIO ESCRIBIR EL GUION (-).
                        font-size: 2em;
                    }

ESTE ES EL RESULTADO DEL EJEMPLO

Este texto tendrá un font size más grande que el resto.
