Unidades em y rem

La unidad em es el tamaño de fuente del contexto. Es variable y depende del contexto, si el elemento es hijo directo de body un em equivale a 16 pixeles, pero si el elemento es hijo de algún otro elemento diferenta a body, la medida para el em se tomara de acuerdo al tamaño de la propiedad font-size de su padre.

Y rem es el tamaño de fuente de root. No importa el contexto porque la raíz siempre es la misma (no es variable).

Ejemplo: Usar la unidad em para establecer el tamaño de fuente de un elemento.

Código HTML usado para el ejemplo.

                        <h1 class="h1">Titulo 
                              <span class="span">de nivel 1</span>
                        </h1>

Código CSS usado para el ejemplo.

                        .h1 { font-size: 3em; }
                              
                        .span {
                              font-size: 1em;
                              color: green;
                        }

Resultado: El tamaño de fuente del h1 es de 3em (48 pixeles) debido a que el contexto de esta no ha cambiado, sigue siendo de 16 pixeles.
Pero el tamaño de fuente del span a pesar de ser de 1em tiene el tamaño de 48 pixeles porque su padre (el h1) ya cambio el contexto del tamaño de la fuente, entonces ahora 1em equivale a 48 pixeles.

Titulo de nivel 1

En el caso de rem que depende de la raiz (html), nada mas necesitamos establecer un tamaño de fuente al elemento html para que cuando se use rem como unidad de medida, esta sea relativa al root y ya no varie.
Ejemplo: supongamos que establecemos el tamaño de fuente de html a 18 pixeles, entonces si al elemento h1 le establecemos el font-size en 2rem este tendrá un tamaño de 32 pixeles, y si al elemento span le establecemos el font-size en 4rem, este tendrá un tamaño de fuente de 72 pixeles.

NOTA: Se recomienda que se use la unidad em en el font-size de un elemento cuando querramos que el padding de este sea proporcional al texto, asi cuando se cambie el tamaño de fuente el padding se ajustara de manera automatica.

La medida rem se recomienda en los margenes o bordes redondeados a fin de que estos sean más consistentes y no cambien debido al contexto.

Direction

La propiedad direction permite cambiar la dirección del texto.
Cabe aclarar que esta propiedad no es muy usada salvo que en alguna ocasion tengamos algún cliente de medio oriente o de china.

Sintaxis

direction: ltr;
direction: rtl;
direction: inherit;
direction: initial;
direction: unset;

Valores

ltr (left to right) El texto se mostrará con orientación de izquierda a derecha.
rtl (right to left) El texto se mostrará con orientación de derecha a izquierda.
inherit Hereda los valores del elemento padre.
initial Establece el valor por default de la propiedad.
unset Quita los valores establecidos.
Ejemplo: Cambiar la dirección del texto de un párrafo

Código HTML usado en el ejemplo.

                        <p class="texto-rtl">Es un simple texto de ejemplo</p>

Código CSS usado para establecer la dirección del texto.

                            .texto-rtl { direction: rtl; }

Resultado: El texto se alinea de derecha a izquierda.

Text indent

La propiedad text-indent permite identar la primera linea de un párrafo. Establece el tamaño de los espacios vacios que se dejaran antes del inicio de la primera linea.

Sintaxis

text-indent: 3mm;

Valores

mm Es la unidad de medida, esta puede ser cualquiera aceptada por CSS (em, px, in, pt).
Esta propiedad tambien admite valores negativos.

Ejemplo: Establecer una identacion a un texto.

Código HTML usado en el ejemplo.

                        <p class="texto-indent">
                              Este es un texto que tiene mas lineas a fin
                              de que se note el indentado del mismo.
                        </p>

Código CSS usado para establecer el indentado al texto.

                            .texto-indent { text-indent: 3em; }

Resultado: Podemos ver el indentado al inicio del texto, este es de 3em (48 pixeles).

Text align

La propiedad text-align establece la alineación horizontal del texto.

Sintaxis

text-align: center;
text-align: left;
text-align: right;
text-align: justify;
text-align: initial;
text-align: inherit;

Valores

center Alinea el texto en el centro del elemento.
left Alinea el texto a la izquierda del elemento (valor por default).
right Alinea el texto a la derecha del elemento.
justify Estira el texto a fin de que cada linea tenga el mismo ancho, excepto la última linea.
initial Alinea el texto con el valor por default.
inherit Hereda el estilo del elemento padre para alinear el texto.
Ejemplo: Establecer la alineacion de un texto usando cualquiera de los valores permitidos.

Código HTML usado para el ejemplo.

                        <p class="texto-align">
                              Lorem ipsum dolor sit, amet consectetur adipisicing elit. Quos reprehenderit quis porro necessitatibus iure dolores aliquam facilis optio                             
                              veritatis, quaerat voluptas. Praesentium fuga dolorem quidem facilis aperiam, ab necessitatibus.Similique ullam magni ipsa aperiam ad ut
                               error, voluptatum tempora sapie
                        </p>

Código CSS usado para establecer la alineación del texto.

                            .texto-align { text-align: center; }

Resultado: Texto alineado al centro del elemento.

Line height

La propiedad line-height especifica la altura de una linea.

Sintaxis

line-height: 3.5;

Valores

3.5 Si al valor no se le especifia una unidad de medida esta tomará el valor y lo multiplicará por el valor del font-size del elemento padre.
Las unidades de medida pueden ser todas las admitidas por CSS.
Tambien se admite los valores inherit, initial y unset.
