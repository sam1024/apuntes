Unidades em y rem

La unidad em es el tamaño de fuente del contexto. Es variable y depende del contexto, si el elemento es hijo directo de body un em equivale a 16 pixeles, pero si el elemento es hijo de algún otro elemento diferenta a body, la medida para el em se tomara de acuerdo al tamaño de la propiedad font-size de su padre.

Y rem es el tamaño de fuente de root. No importa el contexto porque la raíz siempre es la misma (no es variable).

Ejemplo: Usar la unidad em para establecer el tamaño de fuente de un elemento.

Código HTML usado para el ejemplo.

                        <h1 class="h1">Titulo 
                              <span class="span">de nivel 1</span>
                        </h1>

Código CSS usado para el ejemplo.

                        .h1 { font-size: 3em; }
                              
                        .span {
                              font-size: 1em;
                              color: green;
                        }

Resultado: El tamaño de fuente del h1 es de 3em (48 pixeles) debido a que el contexto de esta no ha cambiado, sigue siendo de 16 pixeles.
Pero el tamaño de fuente del span a pesar de ser de 1em tiene el tamaño de 48 pixeles porque su padre (el h1) ya cambio el contexto del tamaño de la fuente, entonces ahora 1em equivale a 48 pixeles.

Titulo de nivel 1

En el caso de rem que depende de la raiz (html), nada mas necesitamos establecer un tamaño de fuente al elemento html para que cuando se use rem como unidad de medida, esta sea relativa al root y ya no varie.
Ejemplo: supongamos que establecemos el tamaño de fuente de html a 18 pixeles, entonces si al elemento h1 le establecemos el font-size en 2rem este tendrá un tamaño de 32 pixeles, y si al elemento span le establecemos el font-size en 4rem, este tendrá un tamaño de fuente de 72 pixeles.

La medida rem se recomienda en los margenes o bordes redondeados a fin de que estos sean más consistentes y no cambien debido al contexto.

Direction

La propiedad direction permite cambiar la dirección del texto.
Cabe aclarar que esta propiedad no es muy usada salvo que en alguna ocasion tengamos algún cliente de medio oriente o de china.

Sintaxis

direction: ltr;
direction: rtl;
direction: inherit;
direction: initial;
direction: unset;

Valores

ltr (left to right) El texto se mostrará con orientación de izquierda a derecha.
rtl (right to left) El texto se mostrará con orientación de derecha a izquierda.
inherit Hereda los valores del elemento padre.
initial Establece el valor por default de la propiedad.
unset Quita los valores establecidos.
Ejemplo: Cambiar la dirección del texto de un párrafo

Código HTML usado en el ejemplo.

                        <p class="texto-rtl">Es un simple texto de ejemplo</p>

Código CSS usado para establecer la dirección del texto.

                            .texto-rtl { direction: rtl; }

Resultado: El texto se alinea de derecha a izquierda.

Text indent

Sintaxis

text-indent: 3mm;

Valores

mm Es la unidad de medida, esta puede ser cualquiera aceptada por CSS (em, px, in, pt).
Esta propiedad tambien admite valores negativos.

Ejemplo: Establecer una identacion a un texto.

                        <p class="texto-indent">

Resultado: Podemos ver el indentado al inicio del texto, este es de 3em (48 pixeles).

Vertical align

La propiedad vertical-align permite alinear un elemento de manera vertical.

Sintaxis

vertical-align: baseline;
vertical-align: sub;
vertical-align: super;
vertical-align: text-top;
vertical-align: text-bottom;
vertical-align: middle;
vertical-align: top;
vertical-align: bottom;
vertical-align: 3em;

Valores

baseline
El elemento es alineado con la linea de base del elemento padre. Es el valor por default.
sub
El elemento está alineado con la línea de base del subíndice del elemento padre.
super
El elemento está alineado con la linea de base del superindice del elemento padre.
text-top
Alinea la parte superior del elemento con la parte superior de la fuente del elemento padre.
text-bottom
Alinea la parte inferior del elemento con la parte inferior de la fuente del elemento padre.
middle
El elemento se coloca en el centro del elemento padre.
top
Alinea la parte superior del elemento y sus descendientes con la parte superior de toda la línea.
bottom
Alinea el fondo del elemento y sus descendientes con el fondo de toda la línea.
3em
Sube o baja un elemento por la longitud especificada. Se permiten valores negativos. Asi como todas las unidades de medida soportadas por CSS.

Tambien soporta los valores initial, inherit y unset.

Ejemplo: Alinear una imagen de manera vertical.

Código HTML usado en el ejemplo.

                        <div class="div-vertical-align">
                              Hola <span class="vertical-align">alineando</span>
                        </div>

Código CSS usado para alinear la imagen de manera vertical.

                            .img-vertical-align { vertical-align: sub; }

Resultado: El elemento span se alinea con el subíndice del elemento padre.

Letter spacing

La propiedad letter-spacing aumenta o disminuye el espacio entre los caracteres de un texto.

Sintaxis

letter-spacing: normal;
letter-spacing: .2rem;

Valores

normal
No hay espacio extra entre los caracteres. A diferencia de un valor de 0, esta palabra clave permite al user agent (navegador) alterar el espacio entre caracteres para justificar el texto. Este es el valor por default.
unidad
Define un espacio extra entre los caracteres además del predeterminado. Los valores negativos están permitidos, pero puede haber limites especificos de implementación. El navegador no puede aumentar o disminuir el espacion entre caracteres para justificar el texto. La unidad puede ser em, rem, cm, mm, in, pt, px.

Tambien soporta los valores initial, inherit y unset.

Ejemplo: Cambiar el espacio entre los caracteres de un texto.

Códig HTML usado en el ejemplo.

                        <p class="letter-spacing">
                              Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatibus accusamus quis, omnis molestiae praesentium quod quisquam eligendi
                              incidunt ad cupiditate aperiam obcaecati excepturi laudantium corrupti architecto sit fuga minus magnam?
                        </p>

Código CSS usado para establecer el espacio entre los caracteres.

                            .letter-spacing { letter-spacing: .4rem; }

Resultado: El espacio entre los caracteres ahora es de .4rem (6.pixeles)

Text decoration line

Sintaxis

Valores

none
El texto no tendrá ninguna decoracion. Es el valor por default, excepto para la etiqueta a.

Text decoration

La propiedad text-decoration es un shorthand de las propiedades text-decoration-line, text-decoration-color y text-decoration-style que especifica la apariencia de las lineas que decoran un texto.

Sintaxis

text-decoration: underline;
text-decoration: overline dotted;
text-decoration: line-through dashed red;
text-decoration: underline overline line-through;

Valores

text-decoration-line
Para el primer caso (underline) solo recibe un valor, el tipo de linea. Este es el único valor requerido.
text-decoration-style
Para el segundo caso (dotted) recibe dos valores, el tipo de linea y el estilo de la misma.
text-decoration-color
Para el tercer caso recibe tres valores, el tipo de linea, el estilo y el color de la misma.
text-decoration-line text-decoration-line text-decoration-line
Para el cuarto caso recibe tres tipos de linea como valor. Esto también es válido.

Los valores por default son none currentcolor solid

Ejemplo Establecer la decoracion de un texto.

Código HTML usado en el ejemplo.

                        <p class="decoration">
                              Lorem ipsum dolor sit amet, consectetur adipisicing elit. Earum possimus expedita maiores quas accusamus, voluptate obcaecati ea eligendi.
                        </p>

Código CSS usado para establecer la decoracion del texto.

                            .decoration { text-decoration: line-through dashed red; }

Resultado: Aplicada la decoración del texto.

NOTA: Las decoraciones de texto se dibujan a través de los elementos de texto de los descendientes. Esto significa que si un elemento especifica una decoración de texto, entonces un elemento hijo no puede eliminar la decoración. Por ejemplo, en la marca de revisión Este texto tiene algunas palabras acentuadas en él. La regla de estilo p { texto-decoración: underline; } haría que el párrafo entero fuera subrayado. La regla de estilo em { text-decoration: none; } no causaría ningún cambio; todo el párrafo seguiría subrayado. Sin embargo, la regla em { text-decoration: overline; } haría que una segunda decoración apareciera en "algunas palabras acentuadas".

Text shadow

La propiedad text-shadow especifica la sombra de un texto.
Acepta una coma como separador para una lista de sombras que serán aplicadas a un texto.

Sintaxis y Valores

text-shadow: 1px 1px 2px black;
Establece los valores: 1 pixel de desplazamiento en el eje x (horizontal), 1 pixel de desplazamiento en el eje y (vertical), 2 pixeles de difuminado (borroso) y el color.
text-shadow: #fc0 1px 0 10px;
La sombra del texto tendrá los valores: color 1 pixel de desplazamiento en el eje x (horizontal), 2 pixeles de desplazamiento en el eje y (vertical) y un difuminado de 10 pixeles.
text-shadow: 5px 5px #558abb;
La sombra del texto tendrá los valores: 5 pixeles de desplazamiento en el eje x (horizontal), 5 pixeles de desplazamiento en el eje y (vertical), el color y no tendrá difuminado.
text-shadow: blue 5px 5px;
La sombra del texto tendrá los valores: color 5 pixeles de desplazamiento en el eje x (horizontal), 5 pixeles de desplazamiento en el eje y (vertical) y no tendrá difuminado.
text-shadow: 5px 5px;
La sombra del texto tendrá los valores: 5 pixeles de desplazamiento en el eje x (horizontal), 5 pixeles de desplazamiento en el eje y (vertical), el color de la sombra será el mismo que tnga el elemento padre como color de frente y no tendrá difuminado.
text-shadow: 1px 1px 2px red, 0 0 1em blue, 0 0 0.2em verde;
La primera sombra del texto tendrá los valores: 1 pixel de desplazamiento en el eje x (horizontal), 1 pixel de desplazamiento en el eje y (vertical), 2 pixeles de difuminado y color rojo, la segunda sombra tendra 0 de desplazamiento en el eje x (horizontal), 0 de desplazamiento en el eje y (vertical) 1em de difuminado y color azul y la tercera sombra tendrá 0 de desplazamiento en el eje x (horizontal), 0 de desplazamiento en el eje y (vertical) 0.2em de difuminado y color verde.
Ejemplo: Establecer la sombra de un texto.

Código HTML usado en el ejemplo.

                        <p class="shadow">
                              Texto con tres sombras.
                        </p>

Código CSS usado para establecer la sombra del texto.

                            .shadow { text-shadow: 1px 1px 2px red, 0 0 1em blue, 0 0 0.2em green; }

Resultado: Se le establecierón tres sombras al texto.

White space

La propiedad white-space especifica el espacio en blanco dentro de un elemento.

Sintaxis y Valores

white-space: normal;
Las secuencias de espacios en blanco se colapsarán en un solo espacio en blanco. El texto se ajustará cuando sea necesario. Este es el valor por defecto.
white-space: nowrap;
Las secuencias de espacios en blanco se colapsarán en un solo espacio en blanco. El texto nunca se ajustará a la siguiente línea. El texto continúa en la misma línea hasta que se encuentra una etiqueta <br>.
No salta de linea.
white-space: pre;
El espacio en blanco es preservado por el navegador. El texto sólo se ajustará a los saltos de línea. Actúa como la etiqueta <pre> en HTML.
white-space: pre-wrap;
El espacio en blanco es preservado por el navegador. El texto se ajustará cuando sea necesario, y en los saltos de línea.
Salta de linea y conserva espacios.
white-space: pre-line;
Las secuencias de espacios en blanco se colapsarán en un solo espacio en blanco. El texto se ajustará cuando sea necesario, y en los saltos de línea. Salta de linea y no conserva espacios.
Ejemplo: Establecer el espacio en blanco a un texto.

Código HTML usado en el ejemplo.

                        <p class="white-space">
                              Lorem ipsum dolor sit amet, consectetur adipisicing elit. Earum possimus expedita maiores quas accusamus, voluptate obcaecati ea eligendi.
                        </p>

Código CSS usado para establecer el espacio en blanco del texto.

                            .white-space { text-shadow: pre; }

Resultado: Al usar el valor pre el texto se verá tal cual está escrito, incluyendo los espacios en blanco que dejemos.

Desbordamiento

El desbordamiento es cuando las palabras de un texto se salen de su contenedor.
para controlar esto existe la propiedad word-break, esta establece si los saltos de línea aparecen dondequiera que el texto desborde su caja de contenido.

Sintaxis y Valores

word-break: normal;
Utiliza las reglas de salto de linea predetermindas. Es el valor por default.
word-break: break-all;
Para evitar el desbordamiento, se deben insertar saltos de palabra entre dos caracteres cualesquiera.
word-break: keep-all;
No se deben utilizar saltos de palabra para textos en chino, japonés o coreano (CJK). El comportamiento de texto sin CJK es el mismo que el de un texto normal.
word-break: break-word;
Para evitar el desbordamiento, las palabras normalmente irrompibles pueden romperse en puntos arbitrarios si no hay puntos de interrupción aceptables en la línea.
No estandarizada.
