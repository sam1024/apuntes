BORDES Y SOMBRAS

Bordes

Para establecer un borde a un elemento HTML se usa la propiedad border, esta puede usarse como shorthand o en propiedades individuales, a continuación se listan algunas.

1. border-style: [estilo] es un shorthand donde estilo puede tomar uno de los siguientes valores dotted, dashed, solid, double, groove, ridge, inset, outset, none y hidden, podemos establecer de uno a cuatro estilos diferentes.
2. border-color [color] el color puede ser establecido por el nombre (red), el código hexadecimal (#3d7a92), RGB (rgb(255, 0, 30)) o RGBA ( rgba(0, 0, 0, .5) ) este ultimo soporta transparencia.
3. border-width [ancho] el ancho puede ser establecido en px, pt, cm, em, etc. o en uno de los tres valores predefinidos thin, medium o thick.
4. border-top [ancho] [estilo] [color] establece el borde superior, este debe recibir tres parámetros (ancho, estilo y color).
5. border-right [ancho] [estilo] [color] establece el borde derecho, este debe recibir tres parámetros (ancho, estilo y color).
6. border-bottom [ancho] [estilo] [color] establece el borde inferior, este debe recibir tres parámetros (ancho, estilo y color).
7. border-left [ancho] [estilo] [color] establece el borde izquierdo, este debe recibir tres parámetros (ancho, estilo y color).
8. border: [grosor] [estilo] [color] este es el shorthand y debe recibir tres parémtros (ancho, estilo y color).

Pero, ¿como se dibuja un borde respecto a la caja?, ¿esta por "encima" de la caja?, ¿está fuera de la caja? o ¿está la mitad dentro y la otra mitad fuera?, saber esto y entenderlo nos ayudara a saber como funcionan las cajas y la maquetación.

NOTA: Para las propiedades 4 a 8 el color y el ancho son opcionales, sino los especificamos estos tomaran el valor por default que establece el navegador.
Ejemplo: Crear una "caja" y establecer las siguientes propiedades borde de 20px punteado de color negro, ancho de 400px, alto de 300px, fondo color rojo, margen izquierdo y derecho automaticos.

Código HTML usado para crear la caja

                    <div class="box"></div>

Código CSS usado para establecer las propiedades a la caja. 

                        .box {
                            border: 20px dotted black;
                            width: 400px;
                            height: 300px; 
                            background: red;
                            margin-left: auto;
                            margin-right: auto;
                        }

Resultado: Como podemos ver con este ejemplo el borde se dibuja por dentro ya que este es una propiedad interna de la caja, la unica propiedad externa es el margin.

NOTA: Al establecer el margen parece que la caja se hace mas "grande", esto se debe al box-sizing, esto puede resultar en comportamientos no deseados y hacer que los demás elementos se "muevan", para evitar esto recordemos siempre establecer mediante el uso del selector universal el box-sizing a border-box de la siguiente manera: (*, *::before, *::after { box-sizing: border-box; }), haciendo esto nos evitaremos muchos problemas a la hora de la maquetación.

Propiedades del border

El borde de una caja está compuesto por cuatro lados (border-top, border-right, border-bottom y border-left) y podemos establecer un estilo distinto para cada uno de ellos. La forma correcta de hacerlo es usando las propiedades de manera individual de la siguiente manera.

border-top-style: [estilo]
border-right-style: [estilo]
border-bottom-style: [estilo]
border-left-style: [estilo]

Si al establecer el border no se especifican valores para width y para color estos tomaran los valores por default, el valor del width será el medium que es su valor por default y el valor del color será el mismo de la caja. usando solo estas propiedades, el mismo tomara un ancho y un color ya que tanto border-color como border-width tienen valores por default, el color del border-color se tomará del color que tenga establecido la caja.

Estas cuatro propiedades tienen un shorthand border-style, al cual le podemos establecer de uno hasta cuatro estilos diferentes, estos aplican en el sentido de las manecillas del reloj, si establecemos cuatro top, right, bottom y left, si establecemos tres uno para top, otro para right y left y el último para el bottom, si establecemos dos uno para top y bottom y otro para right y left, si establecemos solo uno este se aplicará a los cuatro lados.

NOTA: El valor por default de border-color se puede cambiar solo con establecerle un color a la caja con la propiedad color
Ejemplo: Dibujar 4 cajas y establecer los bordes a cada una de ellas usando el shorthand border-style

Código HTML usado para crear la caja

                    <div class="box-1"></div>
                    <div class="box-2"></div>
                    <div class="box-3"></div>
                    <div class="box-4"></div>

Código CSS usado para establecer las propiedades de las cajas.

                        [class^="box-"] {
                            width: 120px;
                            height: 120px;
                            background-color: rgb(13, 27, 219);
                            display: inline-block;
                            margin-top: 5px;
                            margin-right: 5px;
                            margin-bottom: 5px;
                            margin-left: 5px;
                            border-width: 5px;
                        }
                            
                        .box-1 {
                            border-style: double;
                            color: green;
                        }

                        .box-2 { border-style: dashed solid; }

                        .box-3 { border-style: dotted double dashed; }
                        
                        .box-4 { border-style: dotted dashed solid double; }

Resultado: Podemos establecer un estilo diferente para cada borde de la cada caja usando el shorthand.
Debido a que la primera caja tiene la propiedad color establecida en verde el borde toma ese color.
