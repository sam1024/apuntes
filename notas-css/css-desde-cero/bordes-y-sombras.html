BORDES Y SOMBRAS

Bordes

Para establecer un borde a un elemento HTML se usa la propiedad border, esta puede usarse como shorthand o en propiedades individuales, a continuación se listan algunas en forma de shorthand.

1. border-style: [estilo] el estilo puede tomar uno de los siguientes valores dotted, dashed, solid, double, groove, ridge, inset, outset, none y hidden, podemos establecer de uno a cuatro estilos diferentes separados por un espacio, no se necesita establecer ancho ni color al borde ya que estos toman un valor por default.

2. border-color: [color] el color puede ser establecido por el nombre (red), el código hexadecimal (#3d7a92), RGB ( rgb(255, 0, 30) ) o RGBA ( rgba(0, 0, 0, .5) ) este ultimo soporta transparencia, Para que esta propiedad funcione tenemos que establcer un estilo al borde.

3. border-width: [ancho] el ancho puede ser establecido en px, pt, cm, em, etc. o en uno de los tres valores predefinidos thin, medium o thick. Para que esta propiedad funcione tenemos que establcer un estilo al borde.

4. border-top: [ancho] [estilo] [color] establece el borde superior, este debe recibir tres parámetros (ancho, estilo y color), de estos unicamente el estilo es requerido.

5. border-right: [ancho] [estilo] [color] establece el borde derecho, este debe recibir tres parámetros (ancho, estilo y color), de estos unicamente el estilo es requerido.

6. border-bottom: [ancho] [estilo] [color] establece el borde inferior, este debe recibir tres parámetros (ancho, estilo y color), de estos unicamente el estilo es requerido.

7. border-left: [ancho] [estilo] [color] establece el borde izquierdo, este debe recibir tres parámetros (ancho, estilo y color), de estos unicamente el estilo es requerido.

8. border: [grosor] [estilo] [color] recibe tres parámetros (ancho, estilo y color), de estos unicamente el estilo es requerido.

Pero, ¿como se dibuja un borde respecto a la caja?, ¿esta por "encima" de la caja?, ¿está fuera de la caja? o ¿está la mitad dentro y la otra mitad fuera?, saber esto y entenderlo nos ayudara a saber como funcionan las cajas y la maquetación.

NOTA: Para las propiedades 1 y 4 a 8 el color y el ancho son opcionales, sino los especificamos estos tomaran el valor por default.
Para las propiedades 1 y 2 el estilo es requerido, sino se establece el borde no se mostrará.
Ejemplo: Crear una "caja" y establecer el borde con las siguientes caracteristicas: 20px de ancho, punteado y que tenga color negro.

Código HTML usado para crear la caja

                    <div class="box"></div>

Código CSS usado para establecer el borde a la caja. 

                        .box {
                            border: 20px dotted black;
                        }

Resultado: Como podemos ver con este ejemplo el borde se dibuja por dentro ya que este es una propiedad interna de la caja, la unica propiedad externa es el margin.

NOTA: Al establecer el borde parece que la caja se hace mas "grande", esto se debe al box-sizing, esto puede resultar en comportamientos no deseados y hacer que los demás elementos se "muevan", para evitar esto recordemos siempre establecer mediante el uso del selector universal el box-sizing a border-box de la siguiente manera: (*, *::before, *::after { box-sizing: border-box; }), haciendo esto nos evitaremos muchos problemas a la hora de la maquetación.

Propiedades del border

El borde de una caja está compuesto por cuatro lados (border-top, border-right, border-bottom y border-left) y podemos establecer un estilo distinto para cada uno de ellos. La forma correcta de hacerlo es usando las propiedades de manera individual de la siguiente manera.

border-top-style: [estilo], border-right-style: [estilo], border-bottom-style: [estilo], border-left-style: [estilo]

border-top-color: [color], border-right-color: [color], border-bottom-color: [color], border-left-color: [color]

border-top-width: [width], border-right-width: [width], border-bottom-width: [width], border-left-width: [width]

Estas tres propiedades tienen su respectivo shorthand border-style, border-color y border-width a los cuales les podemos establecer de uno hasta cuatro valores diferentes, estos aplican en el sentido de las manecillas del reloj, si establecemos cuatro top, right, bottom y left, si establecemos tres uno para top, otro para right y left y el último para el bottom, si establecemos dos uno para top y bottom y otro para right y left, si establecemos solo uno este se aplicará a los cuatro lados.

NOTA: Como ya vimos, si al establecer el borde con el shorthand border-style no se especifican valores para width y para color estos tomaran los valores por default, el valor del width será el medium y el valor del color será el mismo de la caja.
Cuando usemos los shorthand border-color y border-width tenemos forzosamente que establecer un estilo, de lo contrario el borde no se mostrará.
El valor por default de border-color se puede cambiar solo con establecerle un color a la caja con la propiedad color ( color: [color] )
Ejemplo: Dibujar 4 cajas, a la primera se le establecerá un solo tipo de borde, a la segunda dos, a la tercer tres y a la ultima cuatro usando el shorthand border-style

Código HTML usado para crear la caja

                    <div class="box-1"></div>
                    <div class="box-2"></div>
                    <div class="box-3"></div>
                    <div class="box-4"></div>

Código CSS usado para establecer los bordes de las cajas.

                        .box-1 {
                            border-style: double;
                            color: green;
                        }

                        .box-2 { border-style: dashed solid; }

                        .box-3 { border-style: dotted double dashed; }
                        
                        .box-4 { border-style: dotted dashed solid double; }

Resultado: Podemos establecer un estilo diferente para cada uno de los bordes de las cajas, sin establecer el color y el width.
Debido a que la primera caja tiene la propiedad color establecida en verde el borde toma ese color.

Shorthand border

Vimos que las propiedades border-style, border-color y border-width son shorthands, pero de estos 3 tenemos uno supremo, que es el border, en este se agrupan todas las demas propiedades, style, color y width, su sintaxis es:
border: [width] [style] [color]

Ejemplo: Dibujar una caja y establecer su borde usando el super shorthand border

Código HTML usado para dibujar la caja.

                    <div class="border"></div>

Código CSS usado para establecer el borde la caja.

                        .border {
                            border: 10px solid #e91e63 
                        }

Resultado: Usando el super shorthand border establecimos el ancho, estilo y color del borde.

NOTA: Al igual que el shorthand border-style, el unico valor requerido por este es el estilo, los otros dos son opcionales.
