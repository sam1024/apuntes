PSEUDOCLASES

Las pseudoclases son selectores especiales que dependen del contexto de los elementos HTML como su pocición, si es el primero, el segundo o el último, tambien podría ser en base a la respuesta de algún estado que pudiera ser activado por el usuario, por ejemplo cuando pasa el mouse, cuando da click, cuando la pagina tiene el foco, cuando lo pierde, cuando en un formulario la información es válida o inválida. Entonces es dependiendo del contexto, la posición o el estado es que se van a estar aplicando algunos estilos.

Pseudoclase :root

La pseudo-clase :root de CSS selecciona el elemento raíz de un árbol que representa el documento. En HTML, :root representa el elemento <html> y es idéntico al selector html, excepto que su especificidad es mayor. Este selector es muy usado para declarar variables.

Sintaxis y valores

:root: [valor]
El valor puede ser prácticamente cualquiera, podemos definir color de texto, tamaño de fuente, variables, etc.
Ejemplo: Definir una variable y un tamaño de fuente para el documento HTML 
                    :root {
                        --main-color: #fff;
                        font-size: 20px;
                    }

Pseudoclase :hover

La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor (puntero del mouse).

Los estilos definidos por la pseudoclase :active serán anulados por cualquier pseudo-clase posterior relacionada con el enlace (:link, :visited, o :active) que tenga al menos la misma especificidad. Para darle un estilo apropiado a los enlaces, coloque la regla :hover después de las reglas :link y :visited, pero antes de :active, según lo definido por el orden LVHA: :link — :visited — :hover — :active.

Sintaxis y valores

[selector]:hover { [propiedad]: [valor]; }
El selector, la propiedad y el valor pueden ser cualquiera válidos en CSS
Ejemplo: Cambiar el color de fondo de una caja cuando el puntero del mouse pase sobre ella.

Este es el código HTML usado para pintar la caja.

                    
                        <div class="caja-1"></div>

Este es el código CSS usado para activar la pseudoclase hover

                    
                        .caja-1:hover { background-color: red; }

Resultado: Podemos ver como al pasar el puntero del mouse el color de fondo de la caja cambia.

Podemos ver un ejemplo aquí de como usar la pseudoclase :hover con :active

NOTA: En un dispositivo movil el compartamiento de la pseudoclase hover se activa al momento de hacer touch sobre el elemento. Esto podría ser un problema si se tienen menús que se desplieguen al momento de hacer hover en el elemento, por eso se debe usar con cuidado esta pseudoclase sobre todo en moviles.

Pseudoclase :active

La pseudo-clase :active de CSS representa un elemento (como un botón) que el usuario está activando. Cuando se usa un mouse, la "activación" generalmente comienza cuando el usuario presiona el botón primario del mouse y termina cuando se suelta. La pseudo-clase :active se usa comúnmente en los elementos <a> y <button>, pero también se puede usar en otros elementos.

Los estilos definidos por la pseudoclase :active serán anulados por cualquier pseudoclase posterior relacionada con el enlace (:link, :hover o :visited) que tenga al menos la misma especificidad. Para darle un estilo apropiado a los enlaces, coloque la regla :active después de todas las demás reglas relacionadas con el enlace, tal como lo define el orden LVHA: :link — :visited — :hover — :active.

Sintaxis

selector:active { [propiedad]: [valor]; }
Ejemplo: Cambiar los estilos width y background-color de un boton cuando este este activo.

Este es el código HTML usado en el ejemplo

                    <button class="btn">Dame click</button>

Este es el código CSS usado para establecer los estilos

                    
                        .btn:active {
                              width: 200px;
                              background-color: red;
                        }
