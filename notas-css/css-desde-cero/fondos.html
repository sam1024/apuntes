FONDOS

Background color

La propiedad background-color establece un color de fondo para un elemento HTML.

Sintaxis

background-color: red; Se usa el nombre de el color para establecerlo.
background-color: #bbff00; Se usa el código hexadecimal de el color para establecerlo.
background-color: rgb(8, 89, 126); Se usa el valor rgb de el color para establecerlo.
background-color: rgba(1, 10, 16, .7); Se usa el valor rgba de de el color para establecerlo.
background-color: hsl(50, 33%, 25%); Se usa el valor hsl de el color para establecerlo.
background-color: hsla(50, 33%, 25%, .75); Se usa el valor hsla de el color para establecerlo.
background-color: currentcolor; Se usa el valor que este establecido con la propiedad color del elemento para establecerlo.
Ejemplo: Establecer un color de fondo para una caja, usando cualquiera de los valores permitidos arriba mostrados.

Código HTML usado para dibujar la caja.

                        <div class="backgroundColor"></div>

Código CSS usado para establecer el color de fondo a la caja.

                            .backgroundColor { background-color: #215254; }

Resultado: Para establecer el fondo de este elemento se uso el valor en hexadecimal del color.

NOTA: La propiedad background-color no tiene dimensiones ni posición, este se extiende por todo el espacio que pueda. Para quitar un color de fondo de un elemento podemos usar el valor transparent (background-color: transparent;).

Backgroun image

La propiedad background-image establece una o más imagenes de fondo a un elemento HTML.

Sintaxis

background-image: url("ruta/imagen.jpg"); Se usa la función url para establecer la imagen de fondo, esta recibe como párametro la ruta donde está almacenada dicha imagen.
background-image: url("ruta/imagen1.png"), url("ruta/imagen2.jpg"); Se usan dos funciones url para establecer dos imagenes de fondo, estas van separadas por una coma (,).
background-image: linear-gradient(rgba(0, 0, 255, .5), rgba(0, 0, 255, .5)), url("ruta/imagen.jpg"); Se usa un degradado lineal y una función url para establecer el fondo, estos van se parados por una coma (,).
NOTA: No es necesario el uso de comillas dobles o simples al momento de pasar la ruta a la función url, ya que en CSS no existe el concepto de strings.
Ejemplo: Dibujar una caja y establecerle una imagen de fondo.

Código HTML usado para dibujar la caja.

                        <div class"backgroundImage></div>

Código CSS usado para establecer la imagen de fondo a la caja.

                            .backgroundImage {
                                  background-image: url(../img/terminator1.jpg);
                            }

Resultado

NOTA: En caso de establecer una imagen de fondo que sea más pequeña que el elemento contenedor ó de que haya mucho espacio en el display del dispositivo, está se repetirá, este es su comportamiento por default.

Background repeat

La propiedad background-repeat permite definir como se repetirá la imagen de fondo. Una imagen de fondo puede repetirse de manera horizontal y vertical, o no repetirse.
Como sabemos la imagen cuando es mas pequeña que el espacio disponible por default se repite.

Sintaxis

background-repeat: repeat; la imagen de fondo se repite (comportamiento por default).
background-repeat: repeat-x; la imagen de fondo se repite de manera horizontal.
background-repeat: repeat-y; la imagen de fondo se repite de manera vertical.
background-repeat: round; la imagen de fondo se repite y se estira o comprime para llenar el espacio disponible
background-repeat: space; La imagen de fondo se repite lo más posible sin recortar. La primera y la última imagen están fijadas a cada lado del elemento, y el espacio en blanco se distribuye uniformemente entre las imágenes.
background-repeat: no-repeat; la imagen de fondo no se repite.
background-repeat: repeat space; la imagen de fondo se repite de manera horizontal sin recortar y el espacio en blanco se distribuye, este comportamiento cambia dependiendo del orden en que se coloquen los valores.
Ejemplo: Dibujar una caja, establecerle una imagen de fondo y usar la propiedad background-repeat para establecer el valor round.

Código HTML usado para dibujar la caja

                            <div class"backgroundRepeat></div>

Código CSS usado para establecer la propiedad background-repeat.

                                .backgroundRepeat { background-repeat: round; }

Resultado: Usando el valor round la imagen de fondo se ajusta lo más posible al tamaño de su elemento contenedor.

NOTA: Las propiedades background-image y background-repeat por lo general se usan juntas.
Encontrará ejemplos de su uso aquí.

Background clip

La propiedad background-clip define hasta dónde debe extenderse el fondo (color o imagen) dentro de un elemento. Puede ser content-box, padding-box, border-box o solo el texto del elemento.

Sintaxis

background-clip: content-box; el fondo se extendará solo por el content-box.
background-clip: padding-box; el fondo se extenderá por el content-box y el padding-box.
background-clip: border-box; el fondo se extenderá por el content-box, el padding-box y el border-box (valor por default).
background-clip: text; el fondo se extenderá solo en el texto del elemento.
Ejemplo: Establecer un fondo a una caja usando el valor content-box.

Código HTML usado para dibujar la caja

                            <div class="bgClipCB">
                                  <span class="textCB">content-box</span>
                            </div>

Código CSS usado para establecer el fondo.

                                .bgClipCB { background-clip: content-box; }

Resultado: El color de fondo solo se extiende por el content-box.

Background origin

La propiedad background-origin establece la posición de origen de una imagen de fondo establecida con la propiedad background-image, en otras palabras desde donde comienza esta a dibujarse.
Está muy ligada a la propiedad background-clip.

Sintaxis

background-origin: content-box; la imagen de fondo comienza en la esquina superior izquierda del content-box.
background-origin: padding-box; la imagen de fondo comienza en la esquina superior izquierda del padding-box (valor por default).
background-origin: border-box; la imagen de fondo comienza en la esquina superior izquierda del border-box.
Ejemplo: Dibujar una caja y establecer la posición de la imagen de fondo con la propiedad background-origin

Código HTML usado para dibujar la caja.

                            <div class="bgOriginCB"></div>

Código CSS usado para establecer la posición de la imagen de fondo.

                                .bgOriginCB { background-origin: content-box; }

Resultado: Podemos ver como la imagen de fondo comienza a pintarse desde la esquina superior izquierda del content-box.

NOTA: La propiedad background-origin se usa solo cuando el fondo sea una imagen, porque si el fondo es un color esta es ignorada.

Background size

La propiedad background-size establece el tamaño de la imagen de fondo

Sintaxis

background-size: auto;
background-size: contain;
background-size: cover;
background-size: 30%;
background-size: 200px 100px;

Valores

auto: Es el valor por default, la imagen de fondo se mostrará en su tamaño original. Si el elemento contenedor es más pequeño, solo se mostrará la parte de la imagen que quepa en el.
contain: Escala la imagen lo más grande posible sin recortar o estirar la imagen.
cover: Escala la imagen de fondo para que cubra todo el contenedor, incluso si tiene que estirar la imagen o cortar un poco uno de los bordes de tal manera que no queden espacios en blanco.
Porcentaje: Establece el ancho y el alto de la imagen de fondo en porcentaje del elemento padre. Si sólo se da un valor, el segundo se ajusta a "auto".
Pixeles: Establece el ancho y alto de la imagen de fondo, el primer valor fija la anchura y el segundo la altura. Si solo se establece un valor, el segundo se ajusta a auto.
Ejemplo: Establecer el tamaño de una imagen de fondo de una caja usando dos valores en pixeles

Código HTML usado para dibujar la caja.

                            <div class="bgSizePixels"></div>

Código CSS usado para establecer el tamaño de la imagen de fondo de la caja.

                                .bgSizePixels { background-size: 150px 150px; }

Background position

La propiedad background-position establece la posición de inicio de la imagen de fondo. Por default la imagen de fondo comienza a dibujarse en la esquina superior izquierda (0, 0).

Sintaxis

background-position: top;
background-position: right;
background-position: bottom;
background-position: left;
background-position: center;
background-position: 25% 68%;
background-position: 25px 68px;
background-position: bottom 10px right 20px;
background-position: right 35% bottom 83%;

Valores

top: La imagen de fondo se posicionará arriba y centrada.
right: La imagen de fondo se posicionará a la derecha y centrada.
bottom: La imagen de fondo se posicionará abajo y centrada.
left: La imagen de fondo se posicionará a la izquierda y centrada.
porcentajes El primer valor es la posición horizontal y el segundo para la vertical. La esquina superior izquierda es 0%, 0%. La esquina inferior derecha es el 100%, 100%. Si solo se especifica un valor el segundo será el 50%. El valor por default es 0%, 0%.
unidades El primer valor es la posición horizontal y el segundo para la vertical. La esquina superior izquierda es 0, 0. Las unidades pueden ser px, cm, mm, pt, in, pc, em, etc. Si solo se especifica un valor el segundo será el 50%. El valor por default es 0%, 0%. Se pueden mezclar los porcentajes y las unidades.
bottom 10px right 20px La imagen de fondo se posicionará a 10 pixeles de la parte inferior y a 20 pixeles del lado derecho del contenedor.
right 35% bottom 83% La imagen de fondo se posicionará a 35% de la parte derecha y a 83% de la parte inferior del contenedor.
NOTA: Los valores en unidades o porcentajes. Si el otro valor es left o right, entonces este valor define Y, en relación con el borde superior. Si el otro valor es top o bottom, entonces este valor define X, relativo al borde izquierdo. Si ambos valores son o , entonces el primero define X y el segundo Y.
Ejemplo: Cambiar la posición de la imagen de fondo de una caja, usando 4 valores, dos para los ejes (x, y) (right y bottom) y otros dos en porcentajes.

Código HTML usado para dibujar la caja.

                            <div class="bgSizeRpctBpct">
                                  bottom % right %
                            </div>

Codigo CSS usado para establecer la posición de la imagen de fondo.

                                .bgSizeRpctBpct { background-position: right 35% bottom 50%; }

Resultado: La imagen de fondo se coloca dejando un 35% de espacio libre del lado derecho (eje x) y con el valor bottom 50% se centra manera vertical (eje y).

Background attachment

La propiedad background-attachment establece como se dibuja el fondo respecto al viewport. Si se desplaza al momento de hacer scroll o se queda fija.

Sintaxis

background-attachment: scroll; background-attachment: fixed; background-attachment: local; background-attachment: inherit; background-attachment: initial;

Valores

scroll La imagen de fondo se desplazará con el resto de la página. Es el valor por default.

fixed La imagen de fondo se quedará fija.

local La imagen de fondo se fija en relación al contenido, si el elemento tiene scroll, se desplazará junto al contenido del elemento.

inherit Hereda los valores del elemento padre.

initial Establece el valor por default de la propiedad.

Ejemplo: Establecer una imagen de fondo al body y usar cada uno de los valores permitidos para background-attachment.

Código CSS usado para establecer la posición de la imagen de fondo

                            body { background-attachment: fixed; }

Se puede ver el resultado del ejemplo aquí, y aunque usemos la barra de desplazamiento el fondo se mantiene en la misma posición.

Background multiples

Podemos tener multiples backgrounds, incluso en una sola declaración.

Sintaxis

background-image: url(), url(); Usamos la función url separada por una coma para establecer las imagenes de fondo.
Ejemplo: Dibujar una caja, establecerle tres imagenes de fondo y acomodarlas usando las propiedades background-size, background-repeat y background-position .

Código HTML usado para dibujar la caja.

                        <div class="bgMultiples"></div>

Código CSS usado para establecer las propiedades.

                            .bgMultiples {
                                  background-image: url(../img/flower.jpeg), url(../img/terminator1.jpg),url(../img/ultron.jpg);                                
                                  background-repeat: no-repeat;
                                  background-size: 25%, 50%, 10%;
                                  background-position: bottom left, center, top sright;                                
                            }

Resultado: Podemos ver las 3 imagenes de fondo, cuando usamos el valor no-repeat y solo es uno este se aplica a las 3 imagenes, en el caso del size y el position se establecen 3 valores, estos se aplicarán de acuerdo al orden en que esten las imagenes.

Shorthand background

El shorthand background agrupa las propiedades background-color, background-image, background-position, background-size, background-repeat, background-origin, background-clip y background-attachment en una sola.

Sintaxis

background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment

Los valores van se parados por un espacio. En el caso de la posición y el tamaño se usa la diagonal (/) para seperar los valores, esto es asi debido a que que estas usan las mismas unidades de medida (px, %, etc.).

Ejemplo: Dibujar una caja y establecer dos imagenes de fondo junto con todas las demás propiedades.

Código HTML usado para dibujar la caja.

                        <div class="background"></div>

Código CSS usado para establecer los estilos a las imagenes de fondo de la caja.

                            .background {
                                  background: url(../img/flower.jpeg) 1% 1% / 80% content-box content-box local, 
                                  #5e0606 url(../img/flor-en-agua.jpeg) 100% 98% / 150px 87px;
                                  background-repeat: no-repeat;
                            }

Resultado: Caja con fondo rojo y dos imagenes de fondo.

NOTA: Cuando se establecen dos imagenes de fondo, se deben establecer todos los estilos para cada uno de ellas separados por una coma, y si por alguna razon queremos que la caja tenga tambien un color de fondo este debe establecerse en la ultima declaración de los estilos tal como se muestra en el código CSS.
Podemos evitar repetir el valor no-repeat con solo agregar la propiedad background-repeat al fnal de las declaraciones de estilos.
