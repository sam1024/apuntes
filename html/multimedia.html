Reproducción de video HTML5

ANTES DE HTML5 EL CONTENIDO MULTIMEDIA (AUDIO Y VÍDEO) DE UNA PAGINA SOLO PODÍA REPRODUCIRSE USANDO FLASH, ACTUALMENTE HTML5 PERMITE REPRODUCIR ESTE CONTENIDO DE MANERA DIRECTA, PARA REPRODUCIR UN VIDEO SE USA LA ETIQUETA video ENTRE LOS ATRIBUTOS DE ESTE ELEMENTO PODEMOS ENCONTRAR:
src: INDICA LA RUTA DEL ARCHIVO DE VIDEO QUE SE CARGARÁ O INSERTARÁ EN LA PAGINA.
width: INDICA LA ANCHURA DEL ÁREA DE VISUALIZACIÓN DEL VIDEO EN PIXELES. ESTA PROPIEDAD ES PREFERIBLE QUE SE DEFINA MEDIANTE CSS
height: INDICA LA ALTURA DEL ÁREA DE VISUALIZACIÓN DEL VIDEO EN PIXELES. ESTA PROPIEDAD ES PREFERIBLE QUE SE DEFINA MEDIANTE CSS
autoplay: INDICA SI EL VIDEO COMENZARÁ A REPRODUCIRSE DE FORMA AUTOMATICA TAN PRONTO COMO SEA POSIBLE, SIN DETENERSE PARA TERMINAR LA CARGA DE LOS DATOS. NO ES RECOMENDABLE USAR ESTA OPCIÓN A MENOS QUE EL VIDEO SEA EL UNICO ELEMENTO EN LA PÁGINA.
controls: INDICA SE SE PRESENTARÁN CONTROLES (VOLUMEN, BÚSQUEDA, PAUSAR Y REANUDAR) PARA LA REPRODUCCIÓN DEL VIDEO AL USUARIO.
poster: PERMITE DEFINIR UN POSTER INICIAL PARA EL VIDEO.

EJEMPO DEL USO DE LA ETIQUETA video
CODIGO FUENTE DEL USO DE LA ETIQUETA video 
                <video src="./media/cinema-4D.mp4" controls</video>

TAMBIEN SE PUEDEN PONER LOS CONTROLES DE PLAY Y PAUSE AL VIDEO MEDIANTE EL USUO DE JS.

EJEMPLO DE REPRODUCTOR DE VIDEO USANDO JS

CODIGO FUENTE DE JS PARA EL REPRODUCTOR DE VIDEO 
                    <script>
                        const VIDEO = document.getElementById('video');
                        const PLAY = document.getElementById('play');
                        const PAUSA = document.getElementById('pause');
                        PLAY.addEventListener('click', () => {
                            VIDEO.play();
                        });
                        PAUSA.addEventListener('click', () => {
                            VIDEO.pause();
                        });
                    </script>

Reproducción de audio HTML5

PARA REPRODUCIR AUDIO EN HTML5 SE USA LA ETIQUETA audio, ENTRE SUS ATRIBUTOS PODEMOS DESTACAR LAS SIGUIENTES:
src: INDICA LA RUTA DEL ARCHIVO DE AUDIO QUE SE CARGARÁ O INSERTARÁ EN LA PAGINA.
autoplay: INDICA SI EL AUDIO COMENZARÁ A REPRODUCIRSE DE FORMA AUTOMATICA TAN PRONTO COMO SEA POSIBLE, SIN DETENERSE PARA TERMINAR LA CARGA DE LOS DATOS. NO ES RECOMENDABLE USAR ESTA OPCIÓN A MENOS QUE EL AUDIO SEA EL UNICO ELEMENTO EN LA PÁGINA.
controls: INDICA SE SE PRESENTARÁN CONTROLES (VOLUMEN, BÚSQUEDA, PAUSAR Y REANUDAR) PARA LA REPRODUCCIÓN DEL AUDIO AL USUARIO.
loop: INDICA SI EL AUDIO SE REPETIRA DE NUEVO UNA VEZ QUE LA REPRODUCCIÓN TERMINE.

EJEMPLO DEL USO DEL ELEMENTO audio
CODIGO FUENTE DEL USO DE LA ETIQUETA audio 
                <audio src="./media/alone-Alan Walker.mp3" controls</audio>
