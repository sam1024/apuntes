TABLAS

LAS TABLAS EN HTML SE USAN PARA MOSTRAR DATOS DE MANERA TABULAR, ESTO QUIERE DECIR QUE MOSTRARÁ CONTENIDO ESTRUCTURADO SEMEJANTE A UNA HOJA DE CALCULO, SE USA CUANDO SE QUIERE MOSTRAR IFORMACION COMO CALIFICACIONES O PRODUCTOS EN UN INVENTARIO.
LA SINTAXIS PARA PINTAR UNA TABLA ES: 

            <table>
              <caption>Titulo de la tabla</caption>
              <thead>
                <tr>
                  <td>Nombre</td>
                  <td>Edad</td>
                  <td>Telefono</td>
                </tr>
              </thead>
              <tbody>
                <tr>
                  <td>Sam</td>
                  <td>36</td>
                  <td>4350421</td>
                </tr>
              </tbody>
            </table>

EN CASO DE QUE NECESITEMOS PONER TALVEZ RESULTADOS DE OPERACIONES EN LA TABLA PODEMOS USAR LA ETIEUTA tfoot

          <tfoot>            
            <tr>
              <td>Rsultado 1</td>
              <td>Resultado 2</td>
              <td>Resultado 3</td>
            </tr>
          </tfoot>

EJEMPLO DE UNA TABLA:

Horario de clases
Lunes Martes Miércoles Jueves Viernes
HTML PHP JavaScript Python Css
Java SQL Algortimos Angular PWA

Agrupar celdas

PARA AGRUPAR CELDAS SE USAN LOS ATRIBUTOS COLSPAN Y ROWSPAN, 0COLSPAN FUNCIONA CON LAS COLUMNAS DE UNA TABLA Y ROWSPAN CON LAS FILAS. 

                <table>
                  <caption>Horario de clases</caption>
                  <thead>            
                    <tr>
                      <td>Lunes</td>
                      <td>Martes</td>
                      <td>Miercoles</td>
                      <td>Jueves</td>
                      <td>Viernes</td>
                    </tr>
                  </thead>
                  <tbody>
                    <tr>
                      <td>HTML</td>
                      <td rowspan="2">PHP</td>
                      <td rowspan="2">JavaScript</td>
                      <td rowspan="2">Python</td>
                      <td>CSS</td>
                    </tr>
                    <tr>
                      <td colspan="5">RECESO</td>
                    </tr>
                    <tr>
                      <td>Java</td>
                      <td>SQL</td>
                      <td>Algortimos</td>
                      <td>Angular</td>
                      <td>PWA</td>
                    </tr>
                  </tbody>
                </table>
