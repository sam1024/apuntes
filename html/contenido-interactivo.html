Contenido interactivo en HTML

EL CONTENIDO INTERACTIVO ES AQUEL QUE REACCIONA DE ALGUNA MANERA EN RESPUESTA A ALGO QUE EL USUARIO A HECHO EN LA PÁGINA (CLICK EN BOTON, DESLIZAR LA PANTALLA, ETC.), EN HTML5 TENEMOS 3 ELEMENTOS PARA LA INTERACTIVIDAD CON LA PÁGINA.
details: PERMITE CREAR UN ACORDEON SI USAR JS, ESTO FACILITA MUCHO EL DESARROLLO DE NUESTRA PÁGINA.
dialog: PERMITE CREAR UN CUADRO DE DIALOGO DE TIPO MODAL, PARA QUE ESTE FUNCIONE DE LA MANERA DESEADA DEBE AGREGARSELE EL ATRIBUTO open Y SI QUEREMOS QUE SEMUESTRE AL MOMENTO DE CARGAR ABRIR LAPAGINA SE RECOMIENDA QUE ESTE SE SITUE EN LA PARTE DE ARRIBA, DE PREFERENCIA ANTES DEL BODY.

EJEMPLO DE USO DEL ELEMENTO details
Requerimientos

Requires a computer running an operating system. The computer must have some memory and ideally some kind of long-term storage. An input device as well as some form of output device is recommended.

CÓDIGO FUENTE DE JS

                    <script>
                        const MODAL = document.getElementById('modal');
                        const BTNMODAL = document.getElementById('btn-modal');
                        const CLOSEMODAL = document.getElementById('close-modal');
                        BTNMODAL.addEventListener('click', () => {
                            MODAL.setAttribute('open', 'true');
                        });
                        CLOSEMODAL.addEventListener('click', () => {
                            MODAL.removeAttribute('open');
                        });
                    </script>

Contenido embebido

EL CONTENIDO EMBEBIDO ES AQUEL QUE TRAEMOS DE OTRO SITIO Y LO INCRUSTAMOS EN NUESTRA PÁGINA WEB, EJEMPLO INSERTAR UN VIDEO DE YOUTUBE.
PARA HACERLO SE USA LA ETIQUETA iframe, ESTA REPRESENTA UN CONTEXTO DE NAVEGACIÓN ANIDADA, ES UN DOCUMENTO ANIDADO EN EL DOCUMENTO ACTUAL ENTRE LAS PROPIEDADES MÁS USADAS DE ESTA ESTÁN:
src: INDICA EL EL URL DEL CONTENIDO QUE SE ESTÁ INSERTANDO.
width: INDICA EL ANCHO EN PIXELES DEL FRAME O CONTENEDOR.
height: INDICA EL ALTO EN PIXELES DEL FRAME O CONTENEDOR.
allow: INDICA UNA LISTA DE POLITICAS DE DIRECTIVAS SEPARADAS POR COMA, DONDE CADA ELEMENTO ESPECIFICA UN CONJUNTO DE ORIGENES QUE ESTARÁN PERMITIDOS, LOS VALORES DE ESTA LISTA PUEDEN SER:

  • accelerometer
  • autoplay
  • encrypted-media
  • gyroscope
  • picture-in-picture
allowfullscreen: INDICA QUE SE PERMITIRÁ LA VISTA A PANTALLA COMPLETA DEL VIDEO.
sandbox: PERMITE DESHABILITAR CARACTERISTICAS POTENCIALMENTE PERJUDICIALES O MOLESTAS COMO FOMULARIOS O PROGRAMAS.

EJEMPLO DEL USO DE iframe, INSERTANDO UN VIDEO DESDE YOUTUBE

CÓDIGO FUENTE DEL USO DE iframe, INSERTANDO UN VIDEO DESDE YOUTUBE

                <iframe 
                    width="560"
                    height="315"
                    src="https://www.youtube.com/embed/ed-OyXDxNjM"
                    frameborder="0" 
                    allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" 
                    allowfullscreen>
                </iframe>
