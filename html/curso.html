Las siguientes etiquetas son elementos de linea span: Su uso es parecido al del div, se puede usar para darle estilo a algún elemento, tambien se puede usar como contenedor de textos, no tiene valor semantico. small: Se usa para mostrar un texto en pequeño, como si fuera menos importante este texto no es importante. strong: Se usa para resaltar una parte de un texto, indica que esa parte es muy o talvez las más importante este texto es importante. em: Se usa para poner un texto o parte de el en cursiva y de esta manera resaltarlo, es para darle enfasis este texto es con enfasis. cite: Se usa para hacer citas, como revistas o libros Romm, J. (2008, February 27). The cold truth about climate change. Salon.com. Recuperado de http://www.salon.com. dfn: Se usa para definir o explicar algo, como podrían ser talvez algunas siglas FIFA (Federacion de Futbol Asociacion). code: Se usa paa poner fragmentos de código, aunque si ponemos código HTML dentro de el no funciona, pero si lo hace bien con código JS () => { return a + b; }. data: Se usa para poner datos de algún programa o talvez de un resultado estadistico. br: Se usa para poner un salto de linea. q: Se usa tambien para citas. abbr: Se usa para abreviar del: Se usa para indicar que algo (un texto) se ha borrado, como cuando se pone algo en oferta, aparece el precio anterio con un diagonal (tachado) y enseguida el nuevo. wbr: Se usa cuando una palabra es muy larga, entonces esta etiqueta se pone en medio de la palabra para indicar que en caso de no haber espacio suficiente en pantalla, esta se corte y se muestre en la siguiente linea. i: Este elemento se creo pensando en usarse para diseño, su fin es que algún texto o palabra sea italica texto italica. b: Este elemento se creo pensando en usarse para diseño, su fin es que algún texto o palabra sea en negrita texto en negrita. u: Este elemento se creo pensando en usarse para diseño, su fin es que algún texto o palabra sea subrayada texto subrayado. sup: Se usa para poner un superindice, ejemplo se puede usar para poner una potencia 42 sub: Se usa para poner un subindice, ejemplo se puede poner para poner la formula quimica del agua h2o time: Se usa para poner fechas u horas, pero solo cuando el uso de estas sean muy importante para la app o sitio web. mark: Se usa para marcar o resaltar un texto texto resaltado a: Se usa para los enlaces, estos son la base de toda la web.

Ejemplos: href="./archivo" - Ruta relativa a un archivo que está en la misma pagina. href="../archivo" - Ruta relativa a la carpeta superior. href="../../carpeta/archivo" - Ruta relativa dos carpetas arriba y dentro de una subcarpeta. href="/archivo" - Ruta relativa a la raiz del proyecto, usando esta nos podemos ahorrar el uso de los puntos (.) en las rutas.
